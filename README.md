## OVERVIEW 
Smartparcel is an application that manages parcel deliveries. It allows drivers and users to create, update, list, and delete parcels. 

## AWS SERVICES USED 
DynamoDB - parcel table storage 
S3 - photo storage 
SNS - notifications 
CloudWatch - monitoring logs 
EC2 
Lambda - runs code without managing servers

## Running the app.py 
python3 app.py
